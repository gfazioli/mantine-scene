## Installation

<InstallScript packages="@gfazioli/mantine-scene" />

After installation import package styles at the root
of your application:

```tsx
import '@gfazioli/mantine-scene/styles.css';
```

You can import styles within a layer `@layer mantine-scene`
by importing `@gfazioli/mantine-scene/styles.layer.css`
file.

```tsx
import '@gfazioli/mantine-scene/styles.layer.css';
```

## Usage

`Scene` is a decorative background container that
composes multiple visual layers: gradients, glow
blobs, dot grids, mesh gradients, and noise
textures. Place it behind your content to create
rich, atmospheric backgrounds.

<Demo data={demos.usage} />

## Fullscreen

Set `fullscreen` to cover the entire viewport with
`position: fixed`. The scene sits behind all content
at `z-index: 0` by default (configurable via the
`zIndex` prop).

<Demo data={demos.fullscreen} />

## Gradient

`Scene.Gradient` supports three types: `radial`
(default), `linear`, and `conic`. Customize with
`colors`, `position`, and `angle` props.

<Demo data={demos.gradient} />

## Glow

`Scene.Glow` renders animated floating blobs.
Each blob can be positioned, sized, and animated
independently. Disable animation with
`animate={false}`.

<Demo data={demos.glow} />

## DotGrid

`Scene.DotGrid` creates a repeating dot pattern.
Control `color`, `dotSize`, `spacing`, and
`opacity`.

<Demo data={demos.dotGrid} />

## Mesh

`Scene.Mesh` overlays multiple radial gradients
to simulate a mesh gradient effect. Define color
stops with position and spread.

<Demo data={demos.mesh} />

## Noise

`Scene.Noise` adds a film grain texture via an
inline SVG filter. Adjust `opacity` and `grain`
(base frequency).

<Demo data={demos.noise} />

## Combined

All sub-components can be composed freely.
Layer order follows DOM order.

<Demo data={demos.combined} />

`Scene` supports

you can add styles to any inner element of the
component with `classNames` prop. Follow

documentation to learn more.
